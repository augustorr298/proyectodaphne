 Descripción del Sistema
El Sistema de Control de Asistencias Inteligente (SCAI) es una solución tecnológica diseñada para registrar, consultar y generar reportes de asistencia de estudiantes mediante una aplicación móvil conectada a una base de datos central.

FASE 3 – TOMA DE DECISIONES
Decisión Importante
Elegir entre desarrollar un prototipo funcional con código real o realizar un diseño estructural completo sin programación.
Opciones consideradas
Programar versión básica funcional


Diseñar prototipo conceptual detallado


Decisión Final
Se eligió realizar un prototipo conceptual detallado.
Justificación
Debido al tiempo limitado y recursos básicos, resultaba más viable entregar un diseño bien estructurado que un sistema incompleto.
